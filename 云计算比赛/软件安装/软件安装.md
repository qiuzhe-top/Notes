
#1.jdk安装

###环境配置
```
JAVA_HOME    d:\Java\jdk        // 要根据自己的
CLASSPATH    .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;         //记得前面有个"." 这个是java的大部分包存在的位置
Path         %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
```
###验证安装
```
java -version
javac
```
